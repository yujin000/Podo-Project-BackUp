# 1. 프로젝트 소개
* 개발기간 : 2022.11.01 ~ 2022.11.17
* 개발인원 : 5명
* 구현기능 : 스트리밍 기능 / 티켓팅 / 공지 & 문의 / 사용자 페이지
* CRUD 기능 및 새로운 기능 구현 목표 / 타사와 비교하여 웹에서 음원 스트리밍을 제공하지 않음

### **사용 언어 및 개발환경**
* **Web** : Java, JavaScript, Jquery, HTML5, CSS3
* **DataBase** : Oracle 11
* **Tool** : Eclipse, Visual Studio Code
* **etc** : Git, Tomcat 8.5
